Smart Tips for Data Analysts
Data Cleaning Checklist:
  • Remove exact duplicates (by all key columns).
  • Standardize dates ‚Üí YYYY-MM-DD.
  • Trim spaces, fix case (e.g., UPPER() for codes, PROPER() for names).
  • Impute missing numeric values with median where reasonable.
  • Create a separate Is_Outlier flag instead of deleting rows.
Common Excel Formulas: 
VLOOKUP: =VLOOKUP(A2,Sheet2!A:B,2,FALSE)
INDEX-MATCH: =INDEX(B:B, MATCH(A2, A:A, 0))
IF + AND: =IF(AND(B2>1000,C2="South"),"High","Normal")
Common DAX Patterns: 
Total Sales =
SUMX(Sales, Sales[Quantity] * Sales[Price])

YOY Sales =
CALCULATE([Total Sales], DATEADD('Date'[Date], -1, YEAR))
KPI Thinking:
  • Volume: Total Sales, Orders, Calls, Tickets.
  • Efficiency: AHT, TAT, On-time %, Utilization.
  • Quality: Defect %, Error %, Churn %.
  • Growth: MoM / YoY %, New vs Returning.

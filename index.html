Interview Q&A

1. What is the purpose of DAX?

DAX (Data Analysis Expressions) is used in Power BI for advanced calculations and creating custom measures and calculated columns.

2. Difference between JOIN types?

INNER JOIN returns only matching rows from both tables, LEFT JOIN keeps all left rows, RIGHT JOIN keeps all right rows, and FULL JOIN keeps all rows from both tables.

3. What is EDA?

Exploratory Data Analysis (EDA) is the process of analyzing data to understand patterns, detect anomalies, and test hypotheses using summary statistics and visualizations.
